# nxt-graphics

This repository contains nxtedition core graphics:

+ bodymovin: host graphic for bodymoving graphics
+ ograf: host graphic for [ograf](https://ograf.ebu.io/) graphics
+ rive: host graphic for [rive](https://rive.app/) graphics

The OGraf and Rive hosts expects the entry file (*.ograf.json for ograf, *.riv for rive) URL to be supplied in the `$main` parameter.
